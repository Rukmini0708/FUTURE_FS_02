LeadFlow CRM – Client Lead Management System
LeadFlow CRM is a lightweight, browser-based customer relationship management (CRM) application designed to help track and manage client leads efficiently. It provides a clean interface to organize leads, monitor their progress, and maintain follow-ups—all in one place.
 Features
* Simple Authentication
  Basic login system to restrict access to the CRM dashboard.
* Dashboard Overview
  Get a quick snapshot of your leads with key metrics like total leads, new leads, contacted leads, and conversion rate.
* Lead Management (CRUD)
  * Add new leads
  * View lead details
  * Edit existing leads
  * Delete leads
*Status Tracking
  Track each lead through stages:
* Notes & Follow-ups
  Add and manage notes for each lead to keep track of conversations and next actions.
* Search & Filters
  Easily find leads by name, email, or company, and filter them by status or source.
* Pipeline View (Kanban Style)
  Visualize leads across different stages in a clean pipeline layout.
  * Local Storage Persistence
  All data is stored in the browser using localStorage—no backend required.
# Tech Stack
* **Frontend:** HTML, CSS, JavaScript
* **Storage:** Browser LocalStorage
